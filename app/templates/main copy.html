Pass by value VS Pass by reference

The terms "pass by value" and "pass by reference" are used to describe how variables are passed on.

The purpose of this website is to describe the differences between these two concepts.

What does 'pass by value' means?

Pass by value means that the value of the function parameter is copied into another location of your memory, and when accessing or modifying the variable within your function, only the copy is accessed/modified and the original value is left untouched. Passing by value is how our values are passed on most of the time.

What does 'pass by reference' means?

Passing by reference means that the memory address of the variable (a pointer to the memory location) is passed to the function. This is unlike passing by value, where the value of the variable is passed on.
